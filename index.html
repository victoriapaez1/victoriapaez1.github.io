How to create a personal website using Github

First you will need to create a Github account, if you dont already have one. Once you have created a password and a username, follow the follwing steps:

Step 1: Repository Setup

  • Click on 'Create New Repository'
  • Name your repository as the following: yourusername.github.io
  • Once youv've created your repository name, make sure your account is public for everyone to see

    • Step 2: Create your file and Upload to Github

  • You will need to dowload VS Code on your device
  • Once you have downloaded VS Code, go to'Files' and click on 'New File'
  • Name it 'index.html' and create a basic website structure
  • When you have your code ready, click on 'Upload files'
  • Upload your 'index.html' by dragging the file into your repository
  • Commit the changes

    • Step 3:Check your work:

  • After completing the steps above, navigate yourusername.github.io. It might take a couple of minutues to upload.
